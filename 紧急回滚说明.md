## 💡 关键信息

### 如果服务器显示 200 但 App 显示"暂无结果"

**可能原因：**
1. 图片 base64 编码失败
2. JSON 响应太大
3. App 端解析失败

**排查命令：**
```bash
# 手动测试 API
curl -X POST http://127.0.0.1:5000/health
# 应该返回：{"status": "ok"}
```

---

### 如果服务器显示 500 或其他错误

**可能原因：**
1. Python 代码执行失败
2. SSH 连接断开
3. 服务器资源不足

**排查命令：**
