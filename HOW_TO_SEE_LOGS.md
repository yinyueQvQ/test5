# 📱 如何查看 Xcode 控制台日志

## 🔍 打开控制台的3种方法

### 方法1: 快捷键（最快）
```
按 Command + Shift + Y
```
会在 Xcode 底部打开/关闭控制台

### 方法2: 菜单栏
```
顶部菜单: View → Debug Area → Activate Console
```

### 方法3: 右上角按钮
```
点击 Xcode 右上角的 【⚠️】图标
```

## ⚠️ 如果没有看到任何日志

### 可能原因1: 控制台被过滤了
**解决方法**:
1. 在控制台底部找到搜索框
2. 清空搜索框内容
3. 确认右边的下拉菜单选择 "All Output"

### 可能原因2: 日志被清除了
**解决方法**:
1. 在控制台右键
2. 选择 "Clear Console"（清空之前的日志）
3. 重新点击"开始合成"

### 可能原因3: 应用没有连接到调试器
**解决方法**:
1. 停止应用（Command + .）
2. 重新运行（Command + R）

## 🧪 快速测试步骤

### 1. 在 Xcode 运行应用
```
Command + R
```

### 2. 打开控制台
```
Command + Shift + Y
```

### 3. 清空控制台
```
在控制台内右键 → Clear Console
```

### 4. 在应用中操作
- 选择2个素材
- 选择工艺风格
- 点击"开始合成"

### 5. 观察
- **界面**: 应该立即出现大的加载动画
- **控制台**: 应该有大量 emoji 图标的日志
- **等待**: 30-60秒

## 🚨 常见问题

### Q: 点击按钮后什么都没发生
**A**: 
1. 检查控制台是否有 `🔥🔥🔥 用户点击开始合成` 
2. 如果有，说明代码运行了，继续看后续日志
3. 如果没有，说明按钮点击事件没触发

### Q: 看到"❌ 素材为空"
**A**: 
1. 重新在背包中给素材改名
2. 确保素材显示绿色边框（已选中）
3. 重新点击"开始合成"

### Q: 看到网络错误
**A**: 
1. 检查 Mac 和 iPhone 是否在同一 WiFi
2. 检查 Python 服务器是否运行
3. 检查 IP 地址是否正确（192.168.31.11）
