# 使用指南 - AI图像分割与风格迁移

## 🚀 快速开始

### 第一步：启动应用
1. 在Xcode中打开项目
2. 确保两个模型文件已正确添加到项目中：
   - `is-net-genral-use.mlmodel`
   - `animeganPaprika.mlmodel`
3. 运行应用

### 第二步：选择图片
- 点击**"从相册选择"**来选择现有图片
- 或点击**"拍摄照片"**用相机拍摄新图片
- 📝 **提示**：选择对比度高、主体明确的图片效果更佳

### 第三步：等待AI分割
- 应用会自动使用AI分析您的图片
- 分割完成后，您将看到彩色的区域图
- 不同颜色代表图片中的不同物体或区域

### 第四步：选择目标区域
- 点击下方的**彩色圆圈卡片**来选择您想要的区域
- 可以选择多个区域
- 实时预览会显示您选中的部分
- 📝 **提示**：通常选择主要物体区域效果最佳

### 第五步：开始风格迁移
- 选好区域后，点击**"开始风格迁移"**
- AI将把选中区域转换为动漫风格
- 等待处理完成（可能需要几秒钟）

### 第六步：查看和保存结果
- 查看最终的动漫风格图片
- 点击**"保存图片"**将结果保存到相册
- 或点击**"重新开始"**处理新图片

## 📱 界面说明

### 进度指示器
顶部的进度条显示当前处理步骤：
- 📷 **选择图片** - 选择或拍摄图片
- ✂️ **图像分割** - AI自动分割图像
- 👆 **选择区域** - 选择需要的区域
- 🎨 **风格迁移** - 转换为动漫风格
- ✅ **最终结果** - 查看和保存结果

### 区域选择卡片
- **彩色圆圈**：代表对应区域的颜色
- **选中状态**：显示白色对勾标记
- **区域编号**：显示区域的唯一ID

## 💡 使用技巧

### 选择合适的图片
✅ **推荐**：
- 光线充足的图片
- 主体清晰的照片
- 背景简单的图像
- 对比度较高的图片

❌ **避免**：
- 过度模糊的图片
- 光线太暗的图像
- 背景过于复杂的照片

### 区域选择建议
- **人物照片**：选择人物区域
- **物品照片**：选择主要物品区域
- **风景照片**：选择有趣的景物区域
- **合成效果**：可以尝试选择多个区域

### 获得最佳效果
1. **预处理**：确保图片清晰度足够
2. **精确选择**：仔细选择目标区域
3. **耐心等待**：AI处理需要时间
4. **多次尝试**：不同区域组合产生不同效果

## 🔧 故障排除

### 常见问题解决

**问题**：应用启动后显示模型加载失败
**解决**：
- 检查模型文件是否正确添加到项目中
- 确保文件名完全匹配
- 重新构建项目

**问题**：图片分割结果不理想
**解决**：
- 尝试使用更清晰的图片
- 确保图片中的物体边界明显
- 避免过于复杂的背景

**问题**：无法访问相机或相册
**解决**：
- 检查应用权限设置
- 在iOS设置中允许相机和照片访问
- 重启应用重新请求权限

**问题**：风格迁移处理时间过长
**解决**：
- 这是正常现象，AI处理需要时间
- 确保设备有足够的内存
- 关闭其他不必要的应用

### 性能优化建议
- 定期重启应用释放内存
- 避免处理过大的图片
- 确保设备存储空间充足
- 在处理过程中避免切换应用

## 📋 技术规格

- **支持格式**：JPEG, PNG, HEIC
- **处理尺寸**：统一调整为512x512像素
- **系统要求**：iOS 14.0+
- **设备要求**：支持CoreML的设备

## 🎯 最佳实践

### 拍摄建议
1. **保持稳定**：避免相机抖动
2. **充足光线**：在自然光下拍摄
3. **简洁背景**：选择简单的背景
4. **居中构图**：将主体放在画面中央

### 处理流程
1. **分批处理**：一次处理一张图片
2. **保存原图**：保留原始图片作为备份
3. **实验不同组合**：尝试不同的区域选择
4. **及时保存**：处理完成后及时保存结果

## 📞 获取帮助

2. 检查系统权限设置
3. 尝试重启应用
4. 确保设备和系统版本符合要求
